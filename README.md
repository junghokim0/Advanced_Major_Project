## 실행 방법

프로젝트를 받은 뒤 아래처럼 터미널 3개를 열어서 실행합니다.

### 1) Backend API 서버
```bash
cd backend
npm start
```

### 2) Mock AI 서버
```bash
cd backend
npm run start:ai-mock
```

### 3) Frontend 앱
```bash
cd frontend
npm start
```
